Security and privacy

Everything sent through Language Layer is encrypted in transit using TLS, the same protection your bank's website uses.

Announcement content is never stored. Each message is translated, delivered to connected attendees, and discarded. We keep only delivery records: when a delivery happened, in what language and format, how fast, and whether it met our standards. Records never contain the message text.

Attendees share only a first name or business name and a language preference. No accounts, no passwords, no tracking across events. Presence expires automatically when an event ends.

Questions: contact the event host or the operator of this deployment.
